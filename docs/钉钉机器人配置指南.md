# 钉钉机器人配置指南

本文档介绍如何配置 史丽慧小助理 的钉钉连接器，使其能够通过钉钉接收和发送消息。大约 3 ～ 5 分钟配置完成。

## 前置条件

- 拥有钉钉企业管理员权限
- 钉钉开放平台开发者账号

## 配置步骤

### 1. 创建钉钉企业内部应用

1. 访问 [钉钉开放平台](https://open.dingtalk.com)
2. 登录后，进入「应用开发」→「企业内部开发」
3. 点击「创建应用」
4. 填写应用名称、描述等信息

### 2. 获取应用凭证

1. 在应用详情页，进入「凭证与基础信息」
2. 记录以下信息：
   - **ClientId**：应用的 ClientId
   - **ClientSecret**：应用的 ClientSecret

### 3. 配置应用权限

在「权限管理」页面申请以下权限：

| 权限名称 | 权限码 | 用途 |
|---------|--------|------|
| 企业内机器人发送消息 | `Robot.Message.Send` | 发送消息 |
| 机器人发送单聊消息 | `ChatBot.User.Message.Send` | 发送单聊消息 |
| 机器人发送群聊消息 | `ChatBot.Group.Message.Send` | 发送群聊消息 |
| 获取用户基本信息 | `Contact.User.Read` | 获取用户信息 |

### 4. 配置 Stream 模式

钉钉支持 Stream 模式接收消息，无需配置公网回调地址：

1. 进入「开发管理」→「消息推送」
2. 消息接收模式选择「Stream 模式」
3. 添加订阅事件：
   - `chat_add_user`：用户添加机器人
   - `chat_remove_user`：用户移除机器人
   - `user_modify_org`：用户信息变更

### 5. 在 史丽慧小助理 中配置

1. 打开 史丽慧小助理 设置 → 外部通讯配置
2. 选择「钉钉」标签页
3. 填写配置信息：
   - **ClientId**：应用的 ClientId
   - **ClientSecret**：应用的 ClientSecret
   - **机器人码**（可选）：如果使用自定义机器人码
4. 点击「保存配置」
5. 点击「启动连接器」

### 6. 发布应用

1. 在钉钉开放平台完成应用配置
2. 点击「发布应用」
3. 审核通过后即可使用

## 使用说明

### 单聊使用

1. 在钉钉中搜索并添加机器人
2. 发送任意消息即可开始对话

### 群组使用

1. 将机器人添加到群组
2. 在群组中 @机器人 发送消息

## 配置参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| ClientId | ✅ | 应用的 ClientId |
| ClientSecret | ✅ | 应用的 ClientSecret |
| RobotCode | ❌ | 机器人码，可选 |
| RequirePairing | ❌ | 是否需要配对授权，默认 false |

## 常见问题

### Q: 连接器启动失败？

1. 检查 ClientId 和 ClientSecret 是否正确
2. 确认应用已发布且可用
3. 检查网络是否可以访问钉钉 API

### Q: 收不到消息？

1. 确认消息推送模式已配置为 Stream 模式
2. 确认已订阅相关事件
3. 检查应用权限是否已申请

### Q: 发送消息失败？

1. 检查是否有发送消息的权限
2. 确认用户/群组 ID 正确
3. 检查 access_token 是否有效
